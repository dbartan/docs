## Adım 3: Environment Variables

`.env.example` dosyasını `.env` olarak kopyalayın:

```bash
cp .env.example .env
```

`.env` dosyasını düzenleyin:

```env
# Database
DATABASE_URL="postgresql://user:password@localhost:5432/saha_kontrol"

# NextAuth
NEXTAUTH_SECRET="your-super-secret-key-here"
NEXTAUTH_URL="http://localhost:3000"

# GitHub OAuth (opsiyonel)
GITHUB_CLIENT_ID="your-github-client-id"
GITHUB_CLIENT_SECRET="your-github-client-secret"

# Uploadthing (dosya yükleme)
UPLOADTHING_SECRET="your-uploadthing-secret"
UPLOADTHING_APP_ID="your-uploadthing-app-id"

# App Config
NEXT_PUBLIC_APP_URL="http://localhost:3000"
```

<Warning>
  **Önemli:** `.env` dosyasını asla git'e commit etmeyin. `.gitignore` dosyasında olduğundan emin olun.
</Warning>

### NextAuth Secret Oluşturma

```bash
openssl rand -base64 32
```

Çıktıyı `NEXTAUTH_SECRET` olarak kullanın.

## Adım 4: Prisma Schema

`prisma/schema.prisma` dosyasını güncelleyin:

```prisma
// prisma/schema.prisma
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}

model User {
  id            String         @id @default(cuid())
  email         String         @unique
  name          String
  role          Role           @default(SITE_ENGINEER)
  image         String?
  phone         String?
  title         String?
  projects      ProjectMember[]
  inspections   Inspection[]
  defects       Defect[]
  createdAt     DateTime       @default(now())
  updatedAt     DateTime       @updatedAt
}

enum Role {
  SUPER_ADMIN
  QAQC_MANAGER
  SITE_ENGINEER
  OBSERVER
}

model Project {
  id          String          @id @default(cuid())
  name        String
  code        String          @unique
  description String?
  client      String
  location    String
  startDate   DateTime
  endDate     DateTime?
  status      ProjectStatus   @default(PLANNING)
  members     ProjectMember[]
  documents   Document[]
  inspections Inspection[]
  defects     Defect[]
  createdAt   DateTime        @default(now())
  updatedAt   DateTime        @updatedAt
}

enum ProjectStatus {
  PLANNING
  IN_PROGRESS
  COMPLETED
  ON_HOLD
}

model ProjectMember {
  id        String   @id @default(cuid())
  projectId String
  userId    String
  role      String
  project   Project  @relation(fields: [projectId], references: [id], onDelete: Cascade)
  user      User     @relation(fields: [userId], references: [id], onDelete: Cascade)
  createdAt DateTime @default(now())

  @@unique([projectId, userId])
}

// ... diğer modeller (Document, Inspection, Defect, Report)
```

<Tip>
  Tam Prisma schema için [GitHub repository](https://github.com/your-org/saha-kontrol/blob/main/prisma/schema.prisma)'yi inceleyin.
</Tip>

## Adım 5: Veritabanını Oluşturun

Prisma migration'larını çalıştırın:

```bash
npx prisma db push
```

Seed data ekleyin (opsiyonel):

```bash
npx prisma db seed
```

Prisma Studio ile veritabanını görüntüleyin:

```bash
npx prisma studio
```

<Check>
  Tarayıcınızda `http://localhost:5555` adresinde Prisma Studio açılacaktır.
</Check>

  Tebrikler! İlk projenizi oluşturdunuz. Artık doküman yükleme, kontrol listesi oluşturma ve uygunsuzluk kaydı yapabilirsiniz.

## Sorun Giderme

<AccordionGroup>
  <Accordion title="Database connection hatası" icon="database">
    **Çözüm:**
    - PostgreSQL'in çalıştığından emin olun: `pg_isready`
    - `DATABASE_URL` değişkeninin doğru olduğunu kontrol edin
    - Firewall ayarlarını kontrol edin
  </Accordion>

  <Accordion title="NextAuth hatası" icon="shield-exclamation">
    **Çözüm:**
    - `NEXTAUTH_SECRET` değişkeninin tanımlı olduğundan emin olun
    - `NEXTAUTH_URL` değişkeninin doğru olduğunu kontrol edin
    - Tarayıcı çerezlerini temizleyin
  </Accordion>

  <Accordion title="Dosya yükleme çalışmıyor" icon="file-exclamation">
    **Çözüm:**
    - Uploadthing API anahtarlarının doğru olduğunu kontrol edin
    - Dosya boyutunun limit içinde olduğundan emin olun (max 10MB)
    - Network bağlantınızı kontrol edin
  </Accordion>

  <Accordion title="Tailwind stilleri yüklenmiyor" icon="paintbrush">
    **Çözüm:**
    - `tailwind.config.ts` dosyasını kontrol edin
    - Dev server'ı yeniden başlatın: `npm run dev`
    - `node_modules` klasörünü silin ve yeniden yükleyin
  </Accordion>
</AccordionGroup>
